ABA Sign-In Application

This project ships as a Python web server. The Netlify deployment only hosts this informational page so visitors do not receive a “Page Not Found” error when following the public link.

To interact with the full sign-in experience—including CSV uploads, the admin dashboard, and emergency roll-call reports—please run the application locally or on a platform that supports persistent Python services.

Run locally

  1. Install Python 3.8+.
  2. Clone the repository and open a terminal in the project root.
  3. Start the server:
    python3 app.py
  4. Open http://localhost:8000 in your browser.

Need help? The project README contains detailed setup instructions, or you can reach out to the team that shared the deployment link.

This static notice prevents Netlify visitors from encountering a 404 and clarifies why the interactive features are unavailable here.
